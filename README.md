# HealthConnect

HealthConnect is a simple digital health support web application that helps users check health risks, verify health information, and access emergency services.

## Features

- Symptom Checker
- Mental Health Self Assessment
- Fake Health News Detector
- Emergency Support System
- Voice Assistant
- Multi-language Support (English, Telugu, Hindi)

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run the Project

1. Download or clone the project
2. Open the project folder
3. Open index.html in a web browser

## Project Purpose

This project was created as a student project to demonstrate a digital health support platform that provides basic health guidance and emergency information.

⚠ This project is for educational purposes only and is not a medical diagnosis tool.
